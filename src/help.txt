Usage: tl <option> [args]
  tl start - start working unit
  tl status - get currrent work and time
  tl stop - stop working unit and track time
  tl drop - dismiss current work; forget timestamp
  tl rotate - track time and switch task
  tl change - change current work subject
  tl history - get work history
  tl connect - connect Jira
  tl commit - send last work entry to Jira
  tl help - get help message
